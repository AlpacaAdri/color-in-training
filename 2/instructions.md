# 2: Markdown

Tips: `1`

This assignment reviews your MarkDown knowledge!
You will make a file called `YOUR-NAME.md`, with `YOUR-NAME` replaced with your name.

The document can be about anything, but must have at least:

- 4 Different Headers
- One Image
- One Link
- An Ordered List
- An Unordered List
- 4 Bold Texts
- 4 Italic Texts
- 2 Strikethrough
- 1 Table
- 3 CMD Commands
- 2 Code Blocks
